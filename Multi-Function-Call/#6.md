# Multi Function Call 6

### Question

- 아래와 같은 순서대로 출력 되도록 실행

```
WowWow
^______^
WowWow
Execute me!!!

```

```
int main(int argc, char* argv[]) {
	char buf[20];
	strcpy(buf, argv[1]);
	printf("WowWow\\n");
	goto helpme;
	printf("Execute me!!!\\n");
	exit(0);
helpme:
	printf("^______^ \\n\\n");
}
```
